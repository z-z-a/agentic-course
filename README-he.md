# קורס פיתוח אגנטי

ברוכים הבאים!

עולם הפיתוח משתנה במהירות. כלים מבוססי AI משנים את הדרך שבה אנחנו כותבים, מדבגים ומשחררים קוד — וקצב השינוי רק הולך ומואץ. כמפתחים, התפקיד שלנו מתרחב מעבר לכתיבת קוד מסורתית לעבודה עם סוכנים חכמים שפועלים לצידנו.

קורס זה הוא המבוא שלכם לעולם הפיתוח האגנטי. תלמדו כיצד למנף את הכלים האלה כדי להיות פרודוקטיביים יותר, לכתוב קוד טוב יותר ולהישאר צעד אחד קדימה. מאמצים מוקדמים תמיד נמצאים בעמדת יתרון — ועצם זה שאתם כאן, אתם כבר עושים את הצעד הזה.

זכרו: כל יום הוא הזדמנות ללמוד משהו חדש.

## לפני הקורס

אנא השלימו את ההכנות הבאות **לפני המפגש הראשון**:

1. **התקינו את סביבת הפיתוח המועדפת עליכם** — VS Code, PyCharm, WebStorm, או כל IDE שנוח לכם
2. **התקינו Git** — הורידו מ-[git-scm.com](https://git-scm.com/downloads)
3. **צרו חשבון GitHub** — הירשמו ב-[github.com](https://github.com) אם אין לכם חשבון. תצטרכו מקום לאחסן את הקוד שלכם
4. **התקינו Claude Code** — עקבו אחר [מדריך ההתקנה](setup-guide-he.md)
