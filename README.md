# Agentic Development Course

Welcome!

The software development landscape is evolving rapidly. AI-powered tools are transforming how we write, debug, and ship code — and the pace of change is only accelerating. As developers, our role is expanding beyond traditional coding into orchestrating intelligent agents that work alongside us.

This course is your introduction to the world of agentic development. You'll learn how to leverage these tools to become more productive, write better code, and stay ahead of the curve. Early adopters always have an edge — and by being here, you're already taking that step.

Remember: every day is an opportunity to learn something new.

## Before the Course

Please complete the following setup **before the first meeting**:

1. **Install your preferred IDE** — VS Code, PyCharm, WebStorm, or any IDE you're comfortable with
2. **Install Git** — download from [git-scm.com](https://git-scm.com/downloads)
3. **Set up a GitHub account** — sign up at [github.com](https://github.com) if you don't have one. You will need a place to store your code
4. **Install Claude Code** — follow the [setup guide](setup-guide-en.md)
